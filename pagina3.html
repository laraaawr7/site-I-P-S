SaaS (Software as a Service)

SaaS (Software as a Service) traduzido para o português significa "Software como um Serviço" de maneira literal. Saas é um modelo de computação em núvem o software é entregue aos usuários pela internet, geralmente por meio de um navegador web. No modelo SaaS, os usuários acessam o software online, sem precisar instalá-lo ou gerenciá-lo localmente.
