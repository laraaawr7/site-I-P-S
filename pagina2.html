PaaS (Plataform as a Service)

PaaS (Plataform as a Service) traduzido para o português significa "Plataforma como um Serviço" de maneira literal. Paas é um modelo de computação em núvem que fornece uma plataforma para uso do usuário, esta sendo completa para desenvolvimento execução e gerenciamento de aplicativos, sem que o usuário precise lidar com servidores, armazenamento ou rede.

Exemplos:

Heroku, Google app engine, Microsoft azure app services
