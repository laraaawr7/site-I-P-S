IaaS (Infrastructure as a Service)

IaaS (Infrastructure as a Service) traduzido para o português significa "Infraestrutura como um Serviço" de maneira literal. Iaas é um modelo de computação em núvem que oferece recursos de infraestrutura de TI como serviço pela internet. Isso significa que, esses recursos podem ser alugados em núvem pela internet, em vez de usar servidores físicos.

Exemplos:

Amazon web services, Microsoft azure, Google cloud Plataform
